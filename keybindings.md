

## Application Launchers

| Keybinding | Action |
|-----------|--------|
| `$mod + t` | Open terminal |
| `$mod + d` | Open application menu |
| `$mod + b` | Open web browser |
| `$mod + e` | Open file manager |
| `$mod + q` | Kill focused window |

## Sway Session Management

| Keybinding | Action |
|-----------|--------|
| `$mod + Shift + c` | Reload Sway configuration |
| `$mod + Shift + e` | Exit Sway (with confirmation prompt) |

## Window Focus Movement

### Vim-style keys (configured via variables)

| Keybinding | Action |
|-----------|--------|
| `$mod + $left` | Focus left |
| `$mod + $down` | Focus down |
| `$mod + $up` | Focus up |
| `$mod + $right` | Focus right |

### Arrow keys

| Keybinding | Action |
|-----------|--------|
| `$mod + Left` | Focus left |
| `$mod + Down` | Focus down |
| `$mod + Up` | Focus up |
| `$mod + Right` | Focus right |

## Workspaces

### Switch to Workspace

| Keybinding | Workspace |
|-----------|-----------|
| `$mod + 1` → `$mod + 9` | Workspace 1–9 |
| `$mod + 0` | Workspace 10 |

### Move Window to Workspace

| Keybinding | Workspace |
|-----------|-----------|
| `$mod + Shift + 1` → `$mod + Shift + 9` | Move to workspace 1–9 |
| `$mod + Shift + 0` | Move to workspace 10 |

## Window States

| Keybinding | Action |
|-----------|--------|
| `$mod + f` | Toggle fullscreen |
| `$mod + Shift + f` | Toggle floating mode |
| `$mod + Space` | Toggle focus between floating and tiling |
| `$mod + a` | Focus parent container |

## Scratchpad

| Keybinding | Action |
|-----------|--------|
| `$mod + Shift + -` | Move window to scratchpad |
| `$mod + -` | Show scratchpad window |


## Resize Mode

Enter resize mode:

| Keybinding | Action |
|-----------|--------|
| `$mod + r` | Enter resize mode |

### Resize Controls (in resize mode)

| Keybinding | Action |
|-----------|--------|
| `Left` / `$left` | Shrink width |
| `Right` / `$right` | Grow width |
| `Up` / `$up` | Shrink height |
| `Down` / `$down` | Grow height |
| `Enter` | Exit resize mode |
| `Escape` | Exit resize mode |


## Media & Hardware Keys (Work When Locked)

### Audio

| Keybinding | Action |
|-----------|--------|
| `XF86AudioMute` | Toggle system mute |
| `XF86AudioLowerVolume` | Decrease volume by 5% |
| `XF86AudioRaiseVolume` | Increase volume by 5% |
| `XF86AudioMicMute` | Toggle microphone mute |

### Brightness

| Keybinding | Action |
|-----------|--------|
| `XF86MonBrightnessDown` | Decrease brightness by 5% |
| `XF86MonBrightnessUp` | Increase brightness by 5% |


## Screenshots

| Keybinding | Action |
|-----------|--------|
| `Print` | Take full-screen screenshot |
| `Shift + Print` | Take selected area screenshot |
